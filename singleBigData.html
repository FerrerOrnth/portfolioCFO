Analista de datos

Análisis de Datos y Big Data. Conceptos: datos, información, conocimiento. Fuentes internas y externas para recolección de datos. KDD. Minería de datos. Lógica para programadores. Proposiciones, tablas de verdad, algoritmos, diagramas de flujo. Bases de datos - SQL bases de datos relacionales y no relacionales. Business Intelligence. Stakeholders. Google Data Studio. Programación en Python. Entornos virtuales y Jupiter Notebooks (Google Colaboratory y Deepnote). Bases de datos en Python. GUI. Proyecto CRUD. Estadísticas. Librerías Python para análisis de datos, Numpy, Pandas, Matplotlib, Seaborn.
